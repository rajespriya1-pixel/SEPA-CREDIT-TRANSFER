# SEPA-CREDIT-TRANSFER
# Project Overview
This project analyzes SEPA Credit Transfer transaction data using Excel and Power BI to understand transaction patterns, settlement behavior, amount distribution, and cross-border activity.
# Data Source
Dataset: SEPA Credit Transfer Transactions
Domain: Banking & Financial Services
Year: 2025
Source: GoMask AI
# Objectives
Analyze transaction volume and value
Validate IBAN and transaction status
Study settlement delays and trends
Compare domestic vs cross-border payments
# Tools Used
Excel: Data cleaning, categorization, pivot tables
Power BI: Data modeling, DAX measures, dashboards
# Key Insights
Majority of transactions are successfully settled
Low-value transactions dominate volume
High & very high value transactions contribute most of the total amount
Most transactions settle within 1 business day
Domestic payments dominate, while cross-border payments have higher average value
# Conclusion
SEPA credit transfers are efficient and reliable, with fast settlement and strong dominance of domestic payments. High-value transactions play a critical role in total transaction value despite lower volume.
